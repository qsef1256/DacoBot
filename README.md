## Environment

* `JDA 5.0.0-beta.20`
* `Jakarta EE 10`
* `JPA 3.0`
* `CDI 4.0`
* `Spring Boot 3.1.5`

## Usage

* /도움말

### Setting

* `setting.properties`
* `key.properties`
* `diapedia.yml` (for Diapedia)

해당 파일은 설정 파일을 jar 파일과 같은 디렉터리에 위치시키는 것으로 로드시킬 수 있음
